# Literature search for CDKN2a

## Goal

Conduct a comprehensive literature search for this gene between 2021 and 2025. 

## Implementation

- Search PubMed and Europe PNC for CDKN2A and ARF
- Use MCP servers for both resources. 
- Create a Markdown file RESULTS.md with a paragraph per paper which contains:
    - Paper title
    - Link to the resource
    - A short description
    - DOI
- For each paper validated to ensure that, in fact, this link points to this paper and so on
